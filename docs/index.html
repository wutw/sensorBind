新增传感器

使用说明
1、参数的选项由厂商、传感器型号和传感器类型共同决定的
2、所属网关支持按照业主企业进行筛选
3、多通道的传感器在新增时必须填写通道号。

绑定传感器

使用说明

  1. 1.展示系统中所有的企业列表
  2. 1.选择传感器绑定至该网关上
  3. 2.选择完成后，点击将选择的传感器和网关进行绑定
  4. 3.选择已绑定的传感器后，点击将传感器和网关解绑
  5. 4.不带有唯一标识(UUID)的传感器，只能通过点击“新增”后填入准确信息方能绑定网关，解绑这类传感器后传感器会被直接删除
